# Ex-book
`ex-book`
[Elixir 実践ガイド](https://www.oiax.jp/ex_book1) で勉強しただけのリポジトリ
- あくまで自分用なので、「書いといた方がいいかな？」と思ったトピックのみスクリプトを書いています。
    - `chXX` が XX 章に対応
- 23 章まで `ExUnit` が登場しなかったため、`ex-book/ch23` 以外は全て簡易的な自作テストモジュールを使用しています。
    - `ex-book/test.sh` で全てのテストを実行します。
